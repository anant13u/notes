Different types of stylings in CSS:

  • Indent all elements having the class "Indent-This

    • This element will have an indentation of 20 pixels as this has the class 'Indent-This'

    
    CSS Code:
    .Indent-This {
                  text-indent: 20px;
                 }
  • `div`: a red background, white text, a font size of 32px, center aligned, and bold
    • 
    HTML Code:
    div>Style me via the external method!/div>
    
    CSS Code:
    div {
         background-color: red;
         color: white;
         font-size: 32px;
         text-align: center;
         font-weight: bold;
        }
  • `p`: a green background, white text, and a font size of 18px
    •  
    
    HTML Code:
    p>I would like to be styled with the internal method, please./p>
    
    CSS Code:
    background-color: green;
    color: white;
    font-size: 18px;
  • `button`: an orange background and a font size of 18px
    •  
    
    HTML Code:
    button style="background-color: orange;font-size: 18px;">Inline Method/button> 
    
    No CSS Code for this instance as the Inline method is used to style the element.

  • All odd numbered elements: a light red/pink background, and a list of fonts containing `Verdana` and `DejaVu Sans` with `sans-serif` as a fallback.
    •  
    
    HTML Code:
    p class="odd-numbers">Number 1 - I'm a class!/p
    p class="odd-numbers num3">Number 3 - I'm a class, but cooler!
    p class="odd-numbers">Number 5 - I'm a class!
    
    CSS Code:
    .odd-numbers {
                  background-color: pink;
                  font-family: Verdana, Geneva, Tahoma, sans-serif;
                 }
  • **The second element**: blue text and a font size of 36px 
    • 
    HTML Code:
    div id="num2">Number 2 - I'm one ID./div 
    
    CSS Code:
    #num2 {
           color: blue;
           font-size: 36px;
          }
  • **The fourth element**: a light green background, a font size of 24px, and bold 
    • 
    HTML Code:
    div id="num4">Number 4 - I'm another ID./div 
    
    CSS Code:
    #num4 {
           background-color: lightgreen;
           font-size: 24px;
           font-weight: bold;
          }
  • **The third element**: in addition to the styles for all odd numbered elements, add a font size of 24px
    •  
    
    HTML Code:
    p class="odd-numbers num3">Number 3 - I'm a class, but cooler!/p 
    
    CSS Code:
    .num3 {
           font-size: 24px;
          }
  • **The first element**: a black background and white text
    •  
    
    HTML Code:
    button class="B1">Click Me!/button
    
    CSS Code:
    .B1 {
         background-color: black;
         color: white;
        }
  • **The second element**: a yellow background
    •  
    
    HTML Code:
    button class="B2">No, Click Me!/button
    
    CSS Code:
    .B2 {
         background-color: yellow;
        }
  • **Both elements**: a font size of 28px and a list of fonts containing `Helvetica` and `Times New Roman`, with `sans-serif` as a fallback 
    • 
    HTML Code:
    button class="B1">Click Me!/button    
    button class="B2">No, Click Me!/button
    
    CSS Code:
    .B1, 
    .B2 {
         font-size: 28px;
         font-family: Helvetica, 'Times New Roman', sans-serif;
        }
